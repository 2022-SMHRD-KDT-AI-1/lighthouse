## 프로젝트명
openAPI 기반 서울특별시 장애인을 위한 편의시설 정보 제공 웹사이트
## 주요기능
- 로그인, 로그아웃, 회원가입
- 시설분야와 구 선택에 따라 달라지는 지도
- 지도의 마커를 클릭 시 달라지는 시설 정보와 후기
- 후기 등록
- 후기 열람
## 개발환경
- 사용언어 : Java 1.8 / HTML / CSS / Javascript
- 개발도구 : adobe XD / eclipse
- 서버환경 : tomcat 8.5
- 데이터베이스 : Oracle 11g
- 빌드도구 : maven
